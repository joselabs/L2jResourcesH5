Доступные Голосовые Команды

.menu : Панель управления персонажа
.online : Показывает сколько игроков онлайн в игре
.repair : Восстановление персонажа, находящегося на том же аккаунте
.password : Изменение пароля аккаунта
.sellbuffs : Продажа баффов персонажа
.buffshield : Защищает вас от нежелательных баффов
.combine : Объединяет талисманы одного типа, в 1 с общей продолжительностью!
.dressme : Изменение внешнего вида персонажа Броня/Оружие/Плащ
.expon/.expoff : Включение/Выключение получения опыта
.deposit/.withdraw : Функции хранилища
.hellbound : Отображение уровня острова ада
.teletocl : Телепортация к лидеру клана
.aa/.makeaa : Конвертирование древней адены
.7rb : Отображение статуса квеста по 7 боссам
.stats : Персональные характеристики цели
.notraders : Скрытие/Отображение всех игроков торговцев
.buffanimation : Выключение анимации баффов
