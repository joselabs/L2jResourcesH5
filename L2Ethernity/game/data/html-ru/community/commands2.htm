Доступные Голосовые Команды

.engage : Предложение пожениться
.divorce : Разорвать отношения и стать свободным
.gotolove : Телепортация к жене или мужу
/clanpenalty : Список штрафов, наложенных на клан
/attacklist : Список кланов, которым была объявлена война и они ее не приняли
/underattacklist : Список кланов, которые объявили войну вам, а вы ее не приняли
/warlist : Список кланов, с которыми ведется война
/instancezone : Информацию об оставшемся времени до нового похода в подземелья
/olympiadstat : Победы, поражения и очки Олимпиады на текущий момент
/mybirthday : Дата создания персонажа
/unstuck : Телепорт в город, если вы застряли
/channelcreate : Создание командного канала
/channeldelete : Удаление командного канала
/channelinvite : Приглашение присоединиться к командному каналу
/channelkick : Выгнать из командного канала
/channelleave : Выйти из командного канала
/channelinfo : Посмотреть информацию о текущем командном канале
