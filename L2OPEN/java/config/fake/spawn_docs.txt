warrior- Боевые боты воины.	
dagger - Боевые боты дагерщики.
tank - Боевые боты танки.
archer - Боевые боты лучники.	
mage - Боевые боты маги.	
summoner - Самонеры.
healer- Хилеры (под класс подпадает только Cardinal).
mage_support - ШЕ/ЕЕ/Овер/Варк, т.к. сапорты-маги.
melee_support - свс, бд.


is_spawn_enabled - true/false. Включен ли данный спаун.
description      - сюда можно написать краткое пояснение. Допустим description="спавн на входе в колизей" (данный параметр не обрабатывается)
handicap_attack - в % указывается бонус атаки для этой группы ботов.
handicap_defence - в % указывается бонус защиты для этой группы ботов.
По умолчанию 100%.
Может принимать значения от 5% до 300%.
Можно не вписывать (тогда используется стандартное значение в 100%).

Пример.
<SpawnObject ai_type="tank" is_spawn_enabled="true" description="LOA entrance">
<coord x="-15752" y="209912" z="-3664" count="2" handicap_attack="100" handicap_defence="100"/>
</SpawnObject>
