SITUATION SKILLS PACKAGE:
target = area действует на местность вокруг персонажа (кастуется без таргета)
target = enemy кастуется на врага
target = self кастуется на себя
cond = skill;id;lvl - skill;ид_скилла;лвл_скилла. Особенность - если указать вместо левела скилла "0", то будет применено, если висит эффект данного скилла с ЛЮБЫМ уровнем.
cond = hp;%;0 or 1 - нр цели;процент;условие. 0 - значит ниже, 1 - значит выше
cond = mp;%;0 or 1 - мр цели;процент;условие. 0 - значит ниже, 1 - значит выше
cond = cp;%;0 or 1 - цп цели;процент;условие. 0 - значит ниже, 1 - значит выше

cond применяется в зависимости от target. Если Target = self либо area то cond применяется к себе.
Например <skill target=area cond=hp;70;0> будет юзаться по области вокруг персонажа, но при условии НР ниже 70%.
